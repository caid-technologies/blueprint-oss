# Roadmap

Blueprint OSS is an MVP and research prototype. This roadmap separates **current capabilities** from **future ideas**.

## Current MVP
- Prompt-to-IR pipeline with sequential agents
- Optional image-assisted generation (when a multimodal LLM provider is configured)
- Seed component database (MCUs, sensors, actuators, displays, power)
- Rule-based validation and repair loop
- React Flow schematic visualization
- SVG and Mermaid diagram generation
- BOM, assembly steps, and mechanical notes
- 3D mechanical placement viewer (React Three Fiber)

## Near-term (next iterations)
- Expand component library and pin metadata
- Improve error explanations and repair summaries
- Add clearer user prompts for missing requirements
- More example projects and UI walkthroughs

## Future ideas (not implemented yet)
- PCB-ready exports (footprints, netlists, BOM integration)
- Multi-board or modular system design
- Deeper electrical simulation and timing checks
- Collaborative projects and version history
- Mechanical CAD export formats
